## Instalação

Siga as instruções do repositório original:

```sh
git clone https://github.com/ismaeldouglasdev/caelestia.git ~/.local/share/caelestia
~/.local/share/caelestia/install.fish
```

## ⚠️ Aviso

O script de instalação cria symlinks de todas as configs. Não mova/remova a pasta
do repositório depois de executar o install, senão os apps podem quebrar.
